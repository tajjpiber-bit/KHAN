TradingView Setup Guide

  1. Create an alert on TradingView using our indicator.
  2. Set the **Condition** to the indicator name, and select **`Any alert() function call`**.
  3. Check the **Webhook URL** box and paste your **public tunnel URL** (displayed in your black command window, e.g. `https://your-tunnel.serveousercontent.com/webhook`).
  4. Set the alert message body exactly to: 
    {
  "symbol": "{{ticker}}",
  "timeframe": "{{interval}}",
  "direction": "{{plot_2}}",
  "score": "{{plot_0}}",
  "rules": "{{plot_4}}",
  "smt": "{{plot_3}}",
  "price": "{{close}}",
  "image": "{{image}}"
}
⚠️ Note: If hosting locally, use a utility like ngrok to expose port 8080 to the internet so TradingView can reach your webhook endpoint.
